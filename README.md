# Gig-Escrow-demo
This is a Demo Repository for the MNEE Stablecoin Hackathon submission.

The client with address as : `` has deployed a gig contract with the freelancer address s : `0xE712ABcF226767E24D1243f434aB1638652ac857`. The freelancer has committed a PR request with a new branch as PR #1. The maintainer or the client reviews and merges the PR. As soon as the PR is merged, either the client or the freelancer can trigger the chainlink Oracle to verify the git merge and finally the freelancer will receive funds. Until the PR is merged, the funds are safe and are locked in the escrow contract. The client can even cancel a gig if not satisfied with the work only after 24 hrs of deploying the gig.
